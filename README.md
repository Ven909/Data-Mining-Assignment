Prerequisite Libraries:

1. pandas
2. numpy
3. matplotlib
4. seaborn

Installation:
You can install all dependencies using pip:

```
pip install pandas numpy matplotlib seaborn
```

How to Run:

1. Ensure all .csv files are in their respective folders
2. Run the 2 scripts from your terminal:

```
python task1.py
```

```
python task2-2.py
```
